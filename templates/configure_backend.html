Setup Instructions

Follow these steps to set up your Google Colab GPU backend:

1
Open Google Colab

Go to Google Colab and create a new notebook or upload the provided notebook.

2
Enable GPU Runtime

Go to Runtime → Change runtime type → Hardware accelerator → GPU

3
Run the Backend Service

Execute all cells in the notebook to start the backend service.

4
Get ngrok URL

The notebook will generate a public ngrok URL. Copy this URL.

5
Configure This App

Paste the ngrok URL in the form above and click "Save Configuration".

6
Start Analyzing!

Your app is now connected to GPU-powered backend. Upload images to analyze!

Example ngrok URL:
https://1234-5678-9012-3456.ngrok.io
Troubleshooting:
  • Make sure your Colab notebook is running and hasn't timed out
  • Verify the ngrok URL is correct and accessible
  • Check that the backend service started without errors
  • Ensure you have GPU runtime enabled in Colab
  • Try refreshing the ngrok tunnel if connection fails
